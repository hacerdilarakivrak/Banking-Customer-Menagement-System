# Banking Customer Menagement System

## Project Overview

This project is a full-stack banking web application designed to simulate real-world banking operations. It allows users to manage customers, accounts, deposits, loans, and transactions through an interactive and structured interface.

The application demonstrates the integration of frontend and backend technologies to build a scalable and modular banking system.

---

##  Technologies Used

**Frontend:**

* React (JSX)
* JavaScript
* HTML5 / CSS3
* Vite

**Backend:**

* Node.js
* Express.js

**Other:**

* Git & GitHub

---

## ⚙️ Features

* Customer management system
* Account creation and management
* Deposit operations
* Loan handling system
* Transaction tracking and history
* Modular component-based frontend structure
* Protected routes and secure navigation
